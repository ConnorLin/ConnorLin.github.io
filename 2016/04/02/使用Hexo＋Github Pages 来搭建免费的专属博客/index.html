一、简介

Hexo

Hexo是一个开源的静态博客框架,基于node.js开发,作者是台湾大学生tommy351。

Github

如果是程序员肯定知道，Github是开源代码库以及版本控制系统。在GitHub，用户可以十分轻易地找到海量的开源代码。

Github Pages

Github Pages可以理解为用户编写的、托管在github上的静态网页，你只需将你的改动提交到Github，就可以实现更改你的网页。
